## P0（最高优先，2周）

- [x] T0-1 接入真实数据源：新增 `TelemetryConnector`（日志/APM/监控）和 `CMDBConnector`（服务拓扑/责任人）。

- [x] 会话超时率明显下降，不能出现长期 pending。
- [x] CI 中可自动产出 benchmark 评分并可阻断回归。

## P1（高优先，2-3周）

- [x] T1-1 因果推理层 v2：从“文本归纳”升级到“依赖拓扑 + 证据传播”。

### P1 验收标准
