# Advanced Ops Terminology Design

## Goal

Replace conversational or product-marketing style wording in the advanced area with concise operational terminology.

The target tone is:

- operational
- platform-facing
- concise
- noun-oriented

Not:

- conversational
- instructional in the title itself
- overly productized

## Naming Direction

Use an operations-console style naming system.

Recommended replacements:

- `高级驾驶舱` -> `高级控制台`
- `我现在要做什么` -> `控制台导航`
- `什么时候进哪一页` -> `场景指引`
- `当前平台状态摘要` -> `运行摘要`
- `系统治理与运行控制` -> `运行治理`
- `分析质量评测` -> `质量评估`
- `调查复盘台` -> `会话审计`
- `工具健康与接入控制` -> `工具管理`
- `推荐下一步` -> `处置建议`

## Scope

This change should cover:

- advanced landing page
- governance page
- benchmark page
- investigation workbench page
- tools page

Do not change:

- routes
- feature behavior
- backend contracts

## Success Criteria

The advanced area succeeds if:

1. Page titles read like operational modules.
2. Section headers are concise and noun-oriented.
3. The wording feels consistent across all advanced pages.
